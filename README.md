# Hitster KR (Host) — 화면공유용

유튜브(프리미엄 포함)는 **진행자가 직접 재생**하고,
이 웹은 **곡 뽑기/정답(연도) 공개/플레이어 타임라인/점수**만 관리합니다.

## 실행
```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`

## 사용 흐름
1) 플레이어 추가  
2) (선택) 난이도 필터 지정  
3) `다음 곡 뽑기` → `유튜브에서 검색 열기`로 새 탭에서 재생  
4) 플레이어가 연도 추리 후, 진행자가 “위치” 선택 → `이 위치로 배치`  
5) 필요하면 `정답 공개`

## 곡 리스트 수정
- `src/data/songs.kr.2000_now.json`에서 `year/title/artist/youtubeQuery`를 편집하면 됩니다.
- **연도 정확도는 서비스 준비 단계에서 한 번 검수**해두는 것을 권장합니다. (게임 품질에 직결)

## 룰(기본)
- 맞으면 점수 +1, 틀리면 -1 (0 아래로는 안 내려감)
- 카드 유지/반납 규칙은 App.tsx의 `placeForPlayer`에서 바꾸면 됩니다.
